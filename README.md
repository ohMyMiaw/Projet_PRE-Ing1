# Projet_PRE-Ing1 MEF1
## PETITE SOURIS

## Descripttion

sujet : Projet CavitY-Taskforce

Dans ce jeu, nous contrôlons un dentiste dont le but est simple : sauver le plus de vies. Nous sommes dans notre cabinet et, au fur et à mesure, des vagues de patients arrivent. Pour cela, nous devons les soigner en faisant attention à notre économie et à la patience de nos patients. N'oubliez pas que de grands pouvoirs impliquent de grandes responsabilités.

## Utilisation
// pour lancer le jeu:
- make
- cd build
- ./jeu.exe
